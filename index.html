Course and Program Assessment

GitHub ID: srsonti

Project Overview

The project focuses on analyzing student learning outcomes at both the course and program levels. I developed the ACAT (Assessment Calculation and Aggregation Tool) to automate outcome calculations based on student assessment data, integrating results into comprehensive reports for both individual courses and programs. Additionally, I integrated a learner model agent for generating personalized feedback for students, improving the assessment process.

Sprint Objectives

Sprint Objective
Sprint 1 Set up project structure and configuration files, including integration with assessment data.
Sprint 2 Develop the ACAT Python class for outcome calculation, handling various student performance metrics.
Sprint 3 Build a learner model agent to provide automated feedback for students based on course outcomes.
Sprint 4 Finalize MVP by refining outcome calculation logic, adding more courses, and testing with live data.

Summary of Tasks

  • Designed and implemented the ACAT class in Python to automate outcome calculations for individual courses and entire programs.
  • Implemented Likert score conversion for better interpretation of student performance.
  • Developed a learner model agent to generate personalized feedback based on student performance across multiple courses.
  • Integrated datasets such as student grades and course outcomes to provide aggregated performance reports for institutional assessments.

Additional Key Files:

  • chat_with_gpt.py: Automates interactions with the learner model to provide detailed feedback on student outcomes.
  • student_outcomes: Contains the mapped outcomes for each student.
  • assessment_databases: Houses the data tables used for assessment aggregation.
  • assessment_results: Stores the results of assessments across various courses.
  • assignments: Contains assignment-specific information used in the outcome calculations.
  • course_outcomes: Contains predefined outcomes for each course to align with program objectives.
  • grades: Stores student grades across multiple courses.
  • run_acat.py: Script that runs the ACAT calculations across all integrated data sources.
