Course and Program Assessment

GitHub ID: srsonti

Project Overview

The project focuses on analyzing student learning outcomes at the course level. I developed the ACAT (Assessment Calculation and Aggregation Tool) to automate outcome calculations based on student assessment data. This tool generates comprehensive reports per student using Likert scale conversion (1–5), aligned with course outcomes. Student grade data was converted into Likert scores to support this process, allowing more consistent interpretation across assessments. These Likert-scaled outcomes were then passed to a GPT-based learner model agent that provides individualized feedback for students. Since GPT models are more effective at interpreting qualitative and natural language data, this conversion enabled more meaningful feedback generation. The result is a system that improves transparency and effectiveness in course-level outcome assessments.

Summary of Tasks

  • Developed the ACAT (Assessment Calculation and Aggregation Tool) in Python to automate course-level outcome assessment
  • Mapped student grade data to a standardized Likert scale (1–5) for consistent evaluation
  • Implemented logic to match assessment items to specific course outcomes
  • Integrated a GPT-based learner model agent to generate personalized feedback using Likert-converted results
  • Processed course data from multiple input files, including grades, outcomes, and assignments
  • Generated per-student reports summarizing performance across all course outcomes
  • Tested and validated output across different course datasets for consistency and accuracy

Additional Key Files:

  • chat_with_gpt.py: Automates interactions with the learner model to provide detailed feedback on student outcomes.
  • student_outcomes.py: Contains the mapped outcomes for each student.
  • assessment_databases: Houses the data tables used for assessment aggregation.
  • assessment_results: Stores the results of assessments across various courses.
  • assignments: Contains assignment-specific information used in the outcome calculations.
  • course_outcomes: Contains predefined outcomes for each course to align with program objectives.
  • grades: Stores student grades across multiple courses.
  • run_acat.py: Script that runs the ACAT calculations across all integrated data sources.
